AquaLink™ Hub

The backbone of AquaMesh's long-range IoT network. Efficient, scalable, and field-ready.

Key Features

Mesh Connectivity

Supports multi-hop LoRa mesh routing with cloud relay to web dashboard.

Solar Powered

Integrated solar panel + battery pack for remote deployment with 24/7 uptime.

Plug & Play Setup

Seamless integration with AquaSpectra sensors and AquaView platform.

Deployment & Workflow

Deployment

Mount to a tripod, buoy, or structure. Connect to 1–12 AquaSpectra units with up to 10km range per hop.

Workflow

Sensor data is routed through mesh nodes to AquaLink, which forwards the data to cloud services via cellular or WiFi uplink.

Technical Specs

  • LoRa Range: Up to 10 km per hop
  • Connectivity: Cellular LTE, WiFi fallback
  • Power: 20W solar + 30,000 mAh battery
  • Mounting: Pole, buoy, tripod compatible
  • Interface: AquaView integration, RS-485 sensor ports
  • Durability: Waterproof, field-tested shell
